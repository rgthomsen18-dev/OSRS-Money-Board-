# OSRS Money Board

Single-file flipping & money-making dashboard for Old School RuneScape.
Live prices from the [OSRS Wiki real-time prices API](https://prices.runescape.wiki) — no backend, no build step; all personal data stays in your browser's localStorage.

**Live:** open `index.html` via GitHub Pages · add to Home Screen on iOS for app-like use.
